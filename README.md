# Sheipados — v11 Chrome Fix

Mudança principal:
- Rodapé virou navegação superior em formato de abas, logo abaixo do cabeçalho.
- Isso elimina conflito de clique com a barra inferior do Chrome no iPhone.
- Cabeçalho usa ícone de halter em CSS, então não quebra mesmo se algum arquivo de imagem falhar.
- Ícones do PWA estão também na raiz: apple-touch-icon.png, icon-192.png, icon-512.png e favicon.ico.
- Mantida pasta /icons para compatibilidade.

Importante:
- No GitHub, suba também os arquivos de imagem da raiz.
- Depois de atualizar, abra o link com ?v=11 ou limpe cache/reinstale o atalho.
